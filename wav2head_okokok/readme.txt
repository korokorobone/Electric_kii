# 目的：「okokok.h」を簡単に作るため。
# ファイル構成：
「exe」：変換用実行exeが入っています。
「source」：python3.6で作成した上記exeのソースです。
            (exe 化は pyinstaller 使用しました。)
「readme.txt」：簡単な説明があります。

## 使用方法
・以下の録音wavデータを用意ください。(フリーソフト等で録音してください。)

サンプリング：8bit
サンプリング周波数：8000Hz
フレームデータ数：14000

・「exe」フォルダ内に作成したwavファイルを入れてください。

・「wav2hed_okokok.exe」をクリックください。
※ウイルスバスター使用の方は、反応してしまいます。
　信用していただければファイルを開いてください。
